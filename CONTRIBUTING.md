# Contributing to micrograd-JAX

PRs that improve clarity, fix bugs, or extend functionality are welcome.

## How to Contribute
1. Fork the repo
2. `git checkout -b feature/your-idea`
3. Make changes, run `python -m pytest test/`
4. Open a PR with a clear description

## Areas That Need Help
- Additional JAX transform examples (`scan`, `checkpoint`)
- More neural net architectures in the demo notebook  
- Benchmarks: JAX vs pure NumPy performance comparison
- Documentation improvements

## Code Style
Follow existing style. Keep it minimal and readable — this is a teaching project.
